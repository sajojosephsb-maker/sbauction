# Auction Website

## Features
- User registration and authentication
- Create, update, and delete auctions
- Bid on auctions in real-time
- Search and filter auctions by categories or keywords
- Notifications for winning bids or outbid alerts

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sajojosephsb-maker/sbauction.git
   cd sbauction
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Set up the environment variables in a `.env` file.
4. Run the application:
   ```bash
   npm start
   ```

## Usage
- Navigate to `http://localhost:3000` to access the auction platform.
- Create an account or log in to begin participating in auctions.

## Technology Stack
- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT (JSON Web Tokens)
- Hosting: Heroku (or similar)
